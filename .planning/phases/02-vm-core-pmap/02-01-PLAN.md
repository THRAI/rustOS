  <files>os/src/mm/vm/vm_object.rs, os/src/mm/vm/mod.rs</files>
  <action>
Create os/src/mm/vm/vm_object.rs with:

1. `PageOwnership` enum:
   - `Anonymous` — frame freed on VmObject drop
   - `Cached` — page cache manages lifetime, VmObject only holds reference

2. `OwnedPage` struct wrapping a physical frame + PageOwnership tag.

3. `VmObject` struct:
   - `pages: BTreeMap<u64, OwnedPage>` indexed by page offset within the object
   - `backing: Option<Arc<RwLock<VmObject>>>` — parent in shadow chain
   - `size: usize` — object size in bytes
   - `resident_count: usize` — number of pages currently in this object (not backing)

4. VmObject methods:
   - `new(size: usize) -> Arc<RwLock<Self>>` — create anonymous object
   - `new_shadow(parent: Arc<RwLock<VmObject>>, size: usize) -> Arc<RwLock<Self>>` — insert shadow at chain head (for fork COW)
   - `lookup_page(offset: u64) -> Option<PhysPageNum>` — walk shadow chain: check self.pages first, then backing recursively (iterative loop, not recursion)
   - `insert_page(offset: u64, page: OwnedPage)` — insert into this object's pages map
   - `remove_page(offset: u64) -> Option<OwnedPage>` — remove from this object only
   - `shadow_depth(&self) -> usize` — count chain depth (for debug/testing)

5. Implement `Drop` for VmObject using iterative unwinding:
   ```
   fn drop(&mut self) {
       let mut current = self.backing.take();
       while let Some(arc) = current {
           match Arc::try_unwrap(arc) {
               Ok(rwlock) => {
                   let mut obj = rwlock.into_inner();
                   // Free anonymous pages
                   for (_, page) in obj.pages.drain() {
                       if matches!(page.ownership, PageOwnership::Anonymous) {
                           // free frame
                       }
                   }
                   current = obj.backing.take();
               }
               Err(_) => break, // other references exist, stop
           }
       }
   }
   ```

6. Add `pub mod vm_object;` to os/src/mm/vm/mod.rs and re-export key types.

Use `alloc::sync::Arc` and `spin::RwLock` (already in deps). For frame representation, use the existing `PhysPageNum` from hal. The OwnedPage should hold a `FrameTracker` (from hal::common) for Anonymous pages so frames auto-free, and a raw `PhysPageNum` for Cached pages.

Do NOT touch the existing UserVmArea/UserVmSpace — that's Task 2.

2. `VmArea` struct (new BSD-style, separate from existing UserVmArea):

   - `obj_offset: u64` — offset into the VmObject where this mapping starts
   - `vma_type: VmAreaType` — enum { Anonymous, FileBacked, Stack, Heap, Device }

3. `VmMap` struct:
   - `areas: BTreeMap<VirtAddr, VmArea>` — keyed by start address, O(log n) lookup
   - `pmap: Pmap` reference (will be a type alias, for now use a placeholder generic or PhysPageNum for root page table)

4. VmMap methods:

   - `insert(&mut self, area: VmArea) -> Result<(), VmError>` — check for overlap, insert

5. `VmAreaType` enum: Anonymous, FileBacked, Stack, Heap, Device
