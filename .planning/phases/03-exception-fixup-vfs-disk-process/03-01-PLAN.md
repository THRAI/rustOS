Create a new assembly file with `copy_user_chunk`:
```
// extern "C" fn copy_user_chunk(dst: *mut u8, src: *const u8, len: usize) -> usize
// Returns 0 on success, EFAULT (14) on fault via landing pad.
```

Assembly structure (per CONTEXT.md decisions):
1. **Prologue**: Load current PerCpu pointer from `tp`. Calculate the offset of `pcb_onfault` within PerCpu (count fields: run_queue, timer_wheel, hartid, cpu_id, needs_reschedule — you MUST verify the exact byte offset by checking struct layout). Store the address of the landing pad label into `pcb_onfault`: `la t1, .Lcopy_fault` then `sd t1, OFFSET_ONFAULT(tp)`. **Set sstatus.SUM=1** to permit S-mode access to U-mode pages: `csrs sstatus, (1 << 18)` (SUM is bit 18). Without this, any load/store to user memory traps regardless of PTE permissions.
2. **Copy loop**: Byte-at-a-time loop: `lb t2, 0(a1); sb t2, 0(a0); addi a0, a0, 1; addi a1, a1, 1; addi a2, a2, -1; bnez a2, loop`. (Simple and correct — performance optimization with word copies is deferred.)
3. **Epilogue**: **Clear sstatus.SUM**: `csrc sstatus, (1 << 18)`. Clear `pcb_onfault`: `sd zero, OFFSET_ONFAULT(tp)`. Set `a0 = 0` (success). `ret`.
4. **Landing pad** (`.Lcopy_fault`): **Clear sstatus.SUM**: `csrc sstatus, (1 << 18)`. Clear `pcb_onfault`: `sd zero, OFFSET_ONFAULT(tp)`. Set `a0 = 14` (EFAULT). `ret`.

CRITICAL: The `OFFSET_ONFAULT` constant must match the actual byte offset of `pcb_onfault` in the PerCpu struct. Add a Rust `const_assert!` or `#[cfg(test)]` check that validates this offset. The PerCpu struct contains complex types (RunQueue, IrqSafeSpinLock<TimerWheel>) whose sizes depend on internal layout — use `core::mem::offset_of!` (nightly) or a manual `memoffset`-style calculation.

**1. Create uiomove module** (`kernel/src/mm/uio.rs`):

Implement the uiomove function with page-at-a-time chunking per CONTEXT.md decisions:

```rust
//! uiomove: safe user/kernel data transfer with page-at-a-time chunking.
//!
//! Wraps copy_user_chunk to provide POSIX short-read/short-write semantics:
//! swallows EFAULT if total_copied > 0, returns Err(EFAULT) only if first chunk fails.

use hal_common::{Errno, PAGE_SIZE};

/// Direction of the copy.
#[derive(Debug, Clone, Copy, PartialEq, Eq)]
pub enum UioDir {
    /// Copy from user to kernel (read from user).
    CopyIn,
    /// Copy from kernel to user (write to user).
    CopyOut,
}

/// Result of uiomove: number of bytes successfully copied.
pub struct UioResult {
    pub done: usize,
}

/// Copy `len` bytes between user pointer `user_ptr` and kernel buffer `kern_buf`.
///
/// Copies page-at-a-time: chunk_size = min(remaining, PAGE_SIZE - (ptr & (PAGE_SIZE-1))).
/// Each chunk calls copy_user_chunk. If a chunk faults:
/// - If done > 0: return Ok(UioResult { done }) (short read/write)
/// - If done == 0: return Err(Errno::EFAULT)
pub fn uiomove(
    kern_buf: *mut u8,
    user_ptr: *mut u8,
    len: usize,
    dir: UioDir,
) -> Result<UioResult, Errno> {
    let mut done: usize = 0;
    let mut remaining = len;
    let mut k_off: usize = 0;
    let mut u_off: usize = 0;

    while remaining > 0 {
        // Page-align chunk: never cross a page boundary on the user side
        let user_addr = user_ptr as usize + u_off;
        let page_remain = PAGE_SIZE - (user_addr & (PAGE_SIZE - 1));
        let chunk = core::cmp::min(remaining, page_remain);

        let (dst, src) = match dir {
            UioDir::CopyOut => (
                unsafe { user_ptr.add(u_off) },
                unsafe { kern_buf.add(k_off) },
            ),
            UioDir::CopyIn => (
                unsafe { kern_buf.add(k_off) },
                unsafe { user_ptr.add(u_off) },
            ),
        };

        let ret = copy_user_chunk_call(dst, src as *const u8, chunk);
        if ret != 0 {
            // Fault occurred
            if done > 0 {
                return Ok(UioResult { done });
            } else {
                return Err(Errno::EFAULT);
            }
        }

        done += chunk;
        k_off += chunk;
        u_off += chunk;
        remaining -= chunk;
    }

    Ok(UioResult { done })
}
```

#[cfg(test)]
mod tests {
    use super::*;

    #[test]
    fn full_copy_single_page() {
        FAULT_ABOVE.store(usize::MAX, Ordering::Relaxed);
        let mut kern = [1u8; 128];
        let mut user = [0u8; 128];
        let r = uiomove(kern.as_mut_ptr(), user.as_mut_ptr(), 128, UioDir::CopyOut).unwrap();
        assert_eq!(r.done, 128);
    }

    #[test]
    fn fault_on_first_chunk_returns_efault() {
        FAULT_ABOVE.store(0, Ordering::Relaxed); // fault everything
        let mut kern = [0u8; 64];
        let mut user = [0u8; 64];
        let r = uiomove(kern.as_mut_ptr(), user.as_mut_ptr(), 64, UioDir::CopyIn);
        assert!(r.is_err());
        FAULT_ABOVE.store(usize::MAX, Ordering::Relaxed);
    }

    #[test]
    fn partial_copy_returns_short_read() {
        // Set fault threshold so second page chunk faults
        // Allocate a buffer that spans a "page boundary" in test
        let mut kern = vec![0u8; PAGE_SIZE + 512];
        let mut user = vec![0u8; PAGE_SIZE + 512];
        // Place user buffer at an address, fault above user + PAGE_SIZE
        let user_ptr = user.as_mut_ptr();
        FAULT_ABOVE.store(user_ptr as usize + PAGE_SIZE, Ordering::Relaxed);
        let r = uiomove(kern.as_mut_ptr(), user_ptr, PAGE_SIZE + 512, UioDir::CopyIn).unwrap();
        // Should have copied up to the page boundary
        assert!(r.done > 0);
        assert!(r.done < PAGE_SIZE + 512);
        FAULT_ABOVE.store(usize::MAX, Ordering::Relaxed);
    }

    #[test]
    fn page_chunking_never_crosses_boundary() {
        // Verify chunk sizes by checking that each chunk ends at a page boundary
        // or at the end of the transfer
        FAULT_ABOVE.store(usize::MAX, Ordering::Relaxed);
        let mut kern = vec![0u8; PAGE_SIZE * 3];
        let mut user = vec![0u8; PAGE_SIZE * 3];
        let r = uiomove(kern.as_mut_ptr(), user.as_mut_ptr(), PAGE_SIZE * 3, UioDir::CopyOut).unwrap();
        assert_eq!(r.done, PAGE_SIZE * 3);
    }

    #[test]
    fn zero_length_copy_succeeds() {
        FAULT_ABOVE.store(usize::MAX, Ordering::Relaxed);
        let mut kern = [0u8; 1];
        let mut user = [0u8; 1];
        let r = uiomove(kern.as_mut_ptr(), user.as_mut_ptr(), 0, UioDir::CopyIn).unwrap();
        assert_eq!(r.done, 0);
    }
}
