## Context

**FINDING-01 detail:** `kernel_trap_handler` in `kernel/src/hal/rv64/trap.rs` (line 93-104) checks `pcb_onfault` on access/page faults and redirects to the landing pad. But `user_trap_handler` in `kernel/src/executor/user_task.rs` (line 120) does NOT check `pcb_onfault`. When `copy_user_chunk` is called from an async syscall handler (e.g., SYS_WRITE at line 394), the fault goes through `__user_trap` → `user_trap_handler`, which tries to resolve it as a page fault instead of redirecting to the fixup landing pad. This means a bad user pointer in copy_user_chunk from async context will attempt page fault resolution instead of returning EFAULT.

**FINDING-03 detail:** `copyinstr` (line 620-633) reads user memory byte-by-byte via `unsafe { *((user_ptr + i) as *const u8) }` with no SUM bit, no pcb_onfault, and no fault protection. `sys_openat_async` (line 567-592) does the same. `copyin_argv` (line 637-660) reads user pointer arrays the same way. A bad user pointer will panic the kernel.

**HAL-02:** `set_user_trap_entry()` was dead code at Phase 1 verification. Phase 3 `user_trap.rs` line 71 calls `__trap_return` which sets stvec to `__user_trap` in the assembly. The `trap_return` function at line 77-83 restores stvec to `__kernel_trap` on return. HAL-02 is now satisfied — just needs re-verification.

**VM-09, VM-13, VM-14:** All implemented and working (verified by QEMU integration tests in Phase 3). Phase 2 never got a VERIFICATION.md because the UAT was created but never completed.

In `kernel/src/executor/user_task.rs`, replace all raw user pointer dereferences with fault-safe copies:

**1. Rewrite `copyinstr` (lines 620-633) to use copy_user_chunk:**

```rust
/// Read a NUL-terminated string from user memory. Returns None on fault.
fn copyinstr(user_ptr: usize, max_len: usize) -> Option<String> {
    if user_ptr == 0 {
        return None;
    }
    let mut buf = alloc::vec![0u8; max_len];
    // Copy from user memory using fault-safe copy_user_chunk.
    let rc = unsafe {
        crate::hal::rv64::copy_user::copy_user_chunk(
            buf.as_mut_ptr(),
            user_ptr as *const u8,
            max_len,
        )
    };
    if rc != 0 {
        return None; // EFAULT
    }
    // Find NUL terminator
    let nul_pos = buf.iter().position(|&b| b == 0)?;
    buf.truncate(nul_pos);
    Some(unsafe { String::from_utf8_unchecked(buf) })
}
```

**2. Rewrite `copyin_argv` (lines 637-660) to use copy_user_chunk for pointer reads:**

```rust
fn copyin_argv(user_argv: usize, max_count: usize, max_total: usize) -> alloc::vec::Vec<String> {
    let mut result = alloc::vec::Vec::new();
    if user_argv == 0 {
        return result;
    }
    let mut total = 0usize;
    for i in 0..max_count {
        let ptr_addr = user_argv + i * core::mem::size_of::<usize>();
        let mut str_ptr: usize = 0;
        let rc = unsafe {
            crate::hal::rv64::copy_user::copy_user_chunk(
                &mut str_ptr as *mut usize as *mut u8,
                ptr_addr as *const u8,
                core::mem::size_of::<usize>(),
            )
        };
        if rc != 0 || str_ptr == 0 {
            break;
        }
        if let Some(s) = copyinstr(str_ptr, 256) {
            total += s.len() + 1;
            if total > max_total {
                break;
            }
            result.push(s);
        } else {
            break;
        }
    }
    result
}
```

**3. Rewrite `sys_openat_async` (lines 567-592) to use copyinstr instead of raw pointer loop:**

```rust
async fn sys_openat_async(
    task: &Arc<Task>,
    pathname_ptr: usize,
    flags: usize,
) -> Result<u32, Errno> {
    use crate::fs::fd_table::OpenFlags;

    // Read pathname from user memory using fault-safe copyinstr.
    let path_str = copyinstr(pathname_ptr, 256).ok_or(Errno::EFAULT)?;

    let open_flags = OpenFlags {
        read: true,
        write: (flags & 0x1) != 0 || (flags & 0x2) != 0,
    };

    crate::fs::syscalls::sys_open(&task.fd_table, &path_str, open_flags).await
}
```

All three changes eliminate raw `*((addr) as *const u8)` dereferences of user pointers and replace them with `copy_user_chunk` which sets SUM, installs pcb_onfault, and returns EFAULT on bad addresses.
