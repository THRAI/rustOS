## Objective

Upgrade stub syscalls to real implementations. mmap/munmap/mprotect become proper VM operations. lseek/fstat work through the new FileDescription model. clock_gettime reads hardware time. nanosleep integrates with the executor timer wheel. futex provides user-space synchronization primitives.

## Context

Current state:
- `user_task.rs`: mmap uses a static bump allocator and creates anonymous VMAs. mprotect/munmap are stubs returning 0. clock_gettime returns 0 (stub). lseek/fstat partially implemented. ioctl returns 0 (stub).
- `vm_map.rs`: VmMap has insert, find_area, find_area_mut, fork, remove. No split operation. No top-down address search.
- brk: Already properly implemented with in-place expand/shrink + PTE teardown.
- Timer wheel: `executor::sleep(ms)` exists and works.
- Page table: pmap_enter, pmap_remove, pmap_protect all functional.

User decisions (locked):
- MAP_FIXED: silently overwrite existing mappings (vm_map_delete before insert).
- Address layout: stack at top, mmap top-down (below stack), brk upward from .bss.
- mmap scope: MAP_ANONYMOUS + MAP_PRIVATE only. Private file-backed (MAP_PRIVATE with file): read → page cache RO, write fault → COW. No MAP_SHARED.
- Futex: process-private, keyed by physical address (walk page table). Global HashMap<PhysAddr, Vec<Waker>>. FUTEX_WAIT: atomic compare + park. FUTEX_WAKE: wake N.
- nanosleep: always EINTR on signal (never auto-restart), must write remaining time to user `rem` pointer.
- clock_gettime: rdtime CSR for both CLOCK_MONOTONIC and CLOCK_REALTIME.

    let map_private = flags & 0x02 != 0; // MAP_PRIVATE

        let start = VirtAddr::new(addr & !0xFFF);

        { let mut pmap = task.pmap.lock(); pmap_remove(&mut pmap, start, end); }
        // Free anonymous frames from removed VMAs
        free_removed_frames(removed);
        start.as_usize()

            Some(va) => va.as_usize(),

Futex syscall dispatch:
```rust
// a0=uaddr, a1=op, a2=val, a3=timeout, a4=uaddr2, a5=val3
let op = a1 & 0x7F; // mask out FUTEX_PRIVATE_FLAG
match op {
    0 => { // FUTEX_WAIT
        // Walk page table to get PA of uaddr
        let pa = pmap_extract(&task.pmap.lock(), VirtAddr::new(a0))?;
        let pa_aligned = PhysAddr::new(pa.as_usize() & !(PAGE_SIZE-1));
        let offset = a0 & (PAGE_SIZE - 1);
        let word_pa = PhysAddr::new(pa_aligned.as_usize() + offset);
        // Atomic compare: read *uaddr, if != val → return EAGAIN
        let current = unsafe { (word_pa.as_usize() as *const u32).read_volatile() };
        if current != a2 as u32 { return (-11isize) as usize; } // EAGAIN
        // Park: FutexWaitFuture registers waker in FUTEX_TABLE
        FutexWaitFuture::new(word_pa).await;
        0
    }
    1 => { // FUTEX_WAKE
        let pa = pmap_extract(&task.pmap.lock(), VirtAddr::new(a0))?;
        let offset = a0 & (PAGE_SIZE - 1);
        let word_pa = PhysAddr::new(pa.as_usize() + offset);
        let mut table = FUTEX_TABLE.lock();
        let woken = if let Some(waiters) = table.get_mut(&word_pa) {
            let n = core::cmp::min(a2, waiters.len());
            for w in waiters.drain(..n) { w.wake(); }
            if waiters.is_empty() { table.remove(&word_pa); }
            n
        } else { 0 };
        woken
    }
    _ => (-38isize) as usize, // ENOSYS
}
```
