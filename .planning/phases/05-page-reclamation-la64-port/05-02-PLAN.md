  <action>
Create kernel/src/mm/daemon/page_daemon.rs with BSD-style clock scan algorithm:

1. Maintain global inactive queue (Vec<Arc<VmPage>> or similar) protected by IrqSafeSpinLock
2. page_daemon_task() async function runs infinite loop:
   - Check memory pressure via frame allocator free count
   - If pressure detected: scan up to 64 pages per iteration
   - For each page: try_acquire_exclusive, check refs/dirty/valid
   - If clean and unreferenced: remove from object, free to allocator
   - If dirty: mark for writeback (writeback daemon handles in plan 05-03)
   - yield_now() after each 64-page chunk
3. Sleep 100ms between scans when no pressure

Create kernel/src/mm/daemon/mod.rs:
- Export page_daemon_task

Key: Chunked scanning (64 pages) + yield prevents executor starvation. BSD vm_pageout pattern.
