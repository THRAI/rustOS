Output: writeback_daemon_task() spawned at boot, integrated with page daemon's dirty page queue.

  <action>
Create kernel/src/mm/daemon/writeback_daemon.rs:

1. Maintain global dirty page queue (Vec<Arc<VmPage>>) protected by IrqSafeSpinLock
2. writeback_daemon_task() async function runs infinite loop:
   - Scan dirty queue for pages marked dirty
   - For each dirty page (up to 16 per iteration):
     - try_acquire_exclusive
     - Extract vnode path from page.object (if file-backed)
     - Call fs_write_page(path, offset, pa).await via delegate
     - On success: clear_dirty_all()
     - release_exclusive
   - yield_now() after each 16-page chunk
3. Sleep 500ms between scans

In kernel/src/mm/daemon/mod.rs:
- Add `pub use writeback_daemon::writeback_daemon_task;`

Key: Smaller chunks (16 pages) than page daemon because I/O is slower. BSD vm_pageout_scan pattern.
