<task type="auto">
  <name>Task 2: LA64 trap entry/exit assembly</name>
  <files>kernel/src/hal/la64/trap.S, kernel/src/hal/la64/trap.rs</files>
  <action>
Create kernel/src/hal/la64/trap.S (mechanical translation of rv64 trap.S):
1. __kernel_trap entry: save 32 GPRs to stack (st.d $r0, $sp, 0; st.d $r1, $sp, 8; ...)
2. Save CSRs: ERA (exception return address), PRMD (pre-exception mode), ESTAT (exception status)
3. Call trap_handler(frame: &mut TrapFrame)
4. Restore CSRs and GPRs
5. ertn instruction (LA64 exception return, equivalent to rv64 sret)

Create kernel/src/hal/la64/trap.rs:
1. TrapFrame struct with 32 GPRs + ERA/PRMD/ECFG/ESTAT/kernel_tp
2. trap_handler(frame: &mut TrapFrame) stub: classify ESTAT.Ecode (exception code)
3. Trap dispatch: timer IRQ (Ecode=0), syscall (Ecode=11), page fault (Ecode=1,2,3)

Key: LA64 trap model is similar to MIPS (ESTAT.Ecode for exception type, ERA for return address).
  </action>
  <verify>
    <automated>cargo build --target loongarch64-unknown-none-elf</automated>
  </verify>
  <done>LA64 trap.S and trap.rs exist, trap entry/exit compiles</done>
