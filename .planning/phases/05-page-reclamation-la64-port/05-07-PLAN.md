pub fn trap_handler(frame: &mut TrapFrame) {
    let cause = frame.scause;
    match cause {
        TIMER_IRQ => handle_timer_irq(),
        EXTERNAL_IRQ => handle_external_irq(),
        IPI => handle_ipi(),
        SYSCALL => handle_syscall(frame),
        PAGE_FAULT => handle_page_fault(frame),
        _ => panic!("Unknown trap"),
    }
}

Create kernel/src/hal/la64/irq.rs:

LA64 interrupt model:
1. ECFG CSR (exception config): enable interrupt lines (bit 11 = timer, bit 12 = IPI)
2. ESTAT CSR (exception status): read to determine which interrupt fired
3. HWI (hardware interrupt) lines: 0-12 for various devices

Implement:
1. init_irq(): write ECFG to enable timer (bit 11) and IPI (bit 12)
2. irq_dispatch(estat: u32):
   - Check estat bits to determine interrupt source
   - If bit 11 set: timer IRQ -> call handle_timer_irq()
   - If bit 12 set: IPI -> call handle_ipi()
   - Else: external device IRQ (stub for now)
3. send_ipi(cpu_id: usize): write to IPI CSR to trigger IPI on target CPU

In kernel/src/hal/la64/mod.rs:
- Add `pub mod irq;`
- Call init_irq() in boot sequence

  <files>kernel/src/hal/la64/trap.rs</files>
  <action>
Update kernel/src/hal/la64/trap.rs trap_handler:

1. Read ESTAT CSR from TrapFrame
2. Extract Ecode (exception code, bits 16:21) and IS (interrupt status, bits 0:12)
3. If IS != 0: interrupt occurred -> call irq::irq_dispatch(estat)
4. Else: exception occurred -> match on Ecode:
   - Ecode 0: TLB refill (page fault)
   - Ecode 1: TLB load invalid (page fault)
   - Ecode 2: TLB store invalid (page fault)
   - Ecode 11: Syscall
   - Ecode 8: Address error (access fault)

Key: LA64 combines interrupts and exceptions in ESTAT. Check IS bits first for interrupts, then Ecode for exceptions.

Integration: Timer IRQ -> irq_dispatch -> handle_timer_irq -> timer wheel advance -> waker.wake()
